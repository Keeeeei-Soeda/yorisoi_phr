## 0. はじめに — World IBD Day 直前のこのタイミングで

5/19の World IBD Day、5/16(土) のPOOPキャラバン、そして会報「**見えないものを、見える形に**」企画 — IBD患者さんの“見えない困りごと”を社会に可視化していこうという大阪IBDさまの取り組みに、深く共感しています。

寄り添いPHR（Personal Health Record）は、まさに**患者さん自身の手元で「見えないものを、見える形に」する道具**を目指して開発してきました。本日は、その現状をご覧いただき、大阪IBDの会員さまにとって役立つ形を一緒に考えさせてください。

## 1. 大阪IBDさまへの敬意と、私たちが学んだこと

- **1986年から40年続く、日本で一番長く続くIBD患者会**
- **165名の会員**、すべて補助金なしのボランティア運営
- 「**IBD患者さんを笑顔に**」という一貫した目標
- 「**経験・知恵・知識・情報**」を集約し、行政要望書まで届ける活動

40年の蓄積の上に立って、私たちが新参者として何かを「持ち込む」立場ではないと理解しています。会員さまの**手元の毎日に、ささやかな安心**を足せる道具として、PHRを位置づけたいと思っています。

## 2. IBD患者さんが日常で抱える「見えない困りごと」

| 困りごと | 数字・実態 | 寄り添いPHRが受け持つ部分 |
|---|---|---|
| **トイレ問題** | 患者の **6割**が外出時に常にトイレ場所をチェック | トイレマップ機能 |
| **日常生活への影響** | **7割以上**が「何らかの影響」を実感 | 体調メモ・連続記録 |
| **多剤・長期服薬** | 5-ASA・ステロイド・免疫抑制剤・**生物学的製剤2週/8週** | お薬管理・投与間隔表示 |
| **多科併診の情報分断** | 消化器・大腸肛門外科・皮膚科・眼科・整形外科… | 「みせる」画面（クリニック軸切替） |
| **転院・セカンドオピニオン時の引き継ぎ** | 紹介状なしだと再検査が必要に | 治療歴の可搬化（医療パスポート） |
| **寛解期の「忘れる」リスク** | 調子いい時こそ次の再燃の備えができない | カレンダーで連続記録の可視化 |

## 3. 寄り添いPHRが提供する3つの体験

### 3ボタンのシンプル設計（簡単モード）

```
┌─────────────────────────┐
│  📝  きろくする           │ ← 受診を音声で1分入力
│  📅  ふりかえる           │ ← カレンダーで自分の歩みを見る
│  👤  みせる               │ ← 医師に渡す「医療パスポート」
└─────────────────────────┘
```

- 寛解期の「めんどくさい」を超えるための徹底した最少アクション設計
- ボタン名はひらがな、見出しは普通の日本語（子どもっぽくならない大人向けトーン）
- 「**毎日使うアプリを目指していない**」 — 必要なときに必要な分だけ
- AIは**判定者ではなく代筆者**（診断・重症度判定はしません）

### 設計の起点
- フィードバック源は**患者会の患者さん本人＋同席医師**
- 設計判断で迷ったら「医師の業務効率」ではなく「**患者の続けやすさ**」を選ぶ

## 4. 本日のデモ導線（10分想定）

### パターンA：UC寛解期の30代会社員（仮想ストーリー）
- 2020年に大阪市立総合医療センターで確定診断
- 5-ASA→ステロイドパルス→寛解→リアルダ単独で寛解維持中
- 2023年から大阪IBDの交流会に参加
- 2025年12月に通勤の都合でなにわのクリニックへ転院 → **転院時の引き継ぎが救われた事例**
- 末梢関節炎・口内炎で多科併診（リウマチ科・皮膚科）

→ **見せる画面の「クリニック軸切替」**が活きるストーリー

### パターンB：CD 9年目の20代後半男性（仮想ストーリー）
- 大学2年生でクローン病発症（小腸大腸型）
- 大阪医科薬科大学IBDセンターで9年フォロー
- ヒュミラ→二次無効→ステラーラ（**生物学的製剤2剤目**）
- 軽度の痔瘻・肛門皮垂で大腸肛門外科併診
- エレンタール就寝前1包を9年継続
- 2018年から大阪IBDの会員、定期的に交流会に参加

→ **多剤・栄養療法・腸管外症状の集約**が活きるストーリー

### デモの流れ
1. ホーム3ボタンを見せる
2. 「**きろくする**」音声入力 → AI自動整形（visit/体調メモを判別）
3. 「**ふりかえる**」カレンダーで治療経過の物語を見せる
4. 「**みせる**」でクリニック軸切替 → 「他のクリニックでの話」をAIが1文要約 → コピー/印刷
5. 「**設定**」から詳細モード機能（お薬・検査値・症状・あんしんカード・**患者会・支援情報**）
