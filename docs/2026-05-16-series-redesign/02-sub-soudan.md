# よりそい そうだん — 経験者の声で相談

## 1. 一言で

> **同じ病気を歩いた人の声と、いつもそばにいるAIが、人に話せない不安を受け止める。**

ChatGPTにできない「**過去の自分の言葉と、同じ病気の他者の言葉に接続する**」が差別化の核。
布谷さん「**実は僕の相談相手はChatGPTになってる**」への正面回答。

## 2. 患者の課題

| 課題 | きっかけ | そうだんの答え |
|---|---|---|
| **主治医に聞きにくい** | 「忙しそう」「素人質問が恥ずかしい」 | AIに何度でも聞ける・「いい質問ですね」と返ってくる |
| **家族に話せない** | 「心配かけたくない」「重く受け止められる」 | 保存しない・流れていく「ひとりごと」モード |
| **同じ病気の人の経験を知りたい** | 「自分だけじゃないと知りたい」 | うちあけRAGで実際の患者の声を引いて答える |
| **セカンドオピニオンを考えたい** | 「主治医に申し訳ない」「何を聞けばいいか分からない」 | 自分の記録×経験者の声から相談下書きを生成 |

## 3. 4つの会話モード（疾患・状態で切替）

### 3-1. ひとりごとモード（保存しない）

既存 `public/simple/talk.html` を発展。
質問しない・要約しない・解決策出さない・診断しない・**共感だけ**。
「ここで話したことは、保存されません」を明示。

→ 5/11フィードバック「**ストレスは最大の敵（医学的事実）**」への直接的な答え。

### 3-2. 経験者の声モード（うちあけRAG）

```
あなた: 「最近スキリージに変えたんですが、効いてる感じがしません」
        ↓
   そうだんAI（うちあけRAGで「スキリージ」「UC」検索）
        ↓
そうだんAI: 「同じスキリージに変えた方が3名、うちあけにいらっしゃいます。
            その中で『3週目までは効いてる感じがしなかった』と書かれている方が
            いました。読んでみますか？」
            
            [体験談を読む] [もう少し話す] [主治医ノートに残す]
```

→ 添田白血病版FRD-002 のRAG設計を疾患非依存に汎化。

### 3-3. 診断理解モード（新規発症期向け）

「IBDってどういう病気ですか」「ステロイドの副作用が怖い」など、診断後の初期質問に答える。
医学的に正確 + 患者の言葉で説明 + うちあけから実際の体験談を引く。

### 3-4. セカンドオピニオン準備モード

「主治医に聞きたいことを整理したい」「他の医師に意見を聞きたい」を支援。
自分の記録（てちょうのvisit履歴）+ 経験者の声から、主治医ノートを生成。

→ 既存 `look-back.html` の「主治医ノート候補」と統合。

## 4. AIの振る舞い原則（5/11ドキュメントから継承）

1. **傾聴ファースト** — 受け止め2文・問い1つ／受け止めだけのターンも可
2. **オウム返しと言い換え** — 相手の言葉を具体的に拾う・「そうだったんですね」連発禁止
3. **小さな肯定** — 「気づかれているのが大事ですね」「メモしたの、いいと思います」
4. **自己決定を尊重** — 「○○すべき」ではなく「○○もありますし、××もあります」
5. **無理に終わらせない** — ユーザーの言葉が「保存しよう」「もういい」が出るまで切らない

## 8. 主要画面（image生成候補）

1. **入口** — 「今、話したいことありますか？」（カウンセラー型・無理に答えなくていい）
2. **会話モード選択** — 4つのモードがやさしく並ぶ
3. **経験者の声モード会話** — RAG結果がカードで差し込まれる瞬間
4. **ひとりごとモード** — グレー基調・保存しない明示
5. **セカンドオピニオン準備完了** — 主治医ノートが出来上がる瞬間

## 9. 設計判断（Why）

- **保存しないモードを設けた理由**: 「家族に話せない」「主治医に言えない」のを受け止めるには、保存しない安心が必要
- **うちあけRAGを核にした理由**: ChatGPTの最大の弱点は「同じ病気の他者の経験」を引けないこと。これがうちあけの最大の価値
- **疾患を段階展開する理由**: データ蓄積が不足している疾患でRAGを動かしても「経験者がいない」と返って失望させる
- **「いい質問ですね」を取り入れる理由**: 布谷さんのChatGPT体験「**池上さんみたいに...こっち半分有頂天になって**」が患者の自尊心を回復させる

## 10. やらないこと

- 診断・重症度判定（AIは判定者ではなく代筆者）
- 「○○すべきです」と命令する
- 主治医に取って代わる立場をとる
- うちあけRAGの結果を「正解」として提示（あくまで「他の方の経験」として）
