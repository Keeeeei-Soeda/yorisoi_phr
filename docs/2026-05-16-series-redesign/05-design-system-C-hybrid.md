# PHRシリーズ デザインシステム — 案C: 共通基盤+PHR独自要素（ハイブリッド）

## 0. 方針

「つながる」の柔らかさを**ベース基盤として共有**しつつ、PHRシリーズ独自の「**静かな中の明るさ**」を追加する。
つながる=外向きの庭、PHR=内向きの縁側、というメタファー対比。

## 1. 世界観

- **メタファー**: 縁側・冬の朝の日溜まり・湯気の立つお茶・温かい毛布・本と一緒に休む
- **時間帯**: 冬の朝〜昼・室内
- **温度感**: あたたかい+静かな・絵本+詩集の中間

## 2. キャラクター（つながるキャラ流用+独自ポーズ）

| キャラ | PHRシリーズでの登場の仕方 |
|---|---|
| 🌱 もか（うさぎ） | 入口AI・**縁側でお茶を飲んでいる**ポーズ |
| 🍞 ぱお（くま） | きろくの空状態・**本に枝折を挟む**ポーズ |
| 🌿 そら（猫） | そうだんの会話相手・**毛布に包まる**ポーズ |
| 🌸 ふう（小鳥） | てちょうのお便り運び・**窓辺で外を見る**ポーズ |

→ キャラは同じ・ポーズだけPHR独自に「**家の中で休んでいる**」シーン。

## 3. カラー

つながるの基盤を継承しつつ、PHR独自に冬要素を追加：

```css
:root {
  /* つながるから継承 */
  --peach-50:        #FFF1E8;
  --peach-100:       #FFD9C7;
  --peach-200:       #FFB89A;
  --peach-600:       #E66A45;
  --matcha-100:      #E8F0D9;
  --matcha-600:      #6E8F4F;
  --lavender-200:    #E0D6E8;
  --bg-cream:        #FFFBF5;

  /* PHRシリーズ独自追加 */
  --tea-300:         #C9A876;  /* お茶の色 */
  --tea-600:         #8F6B3F;
  --winter-blue-100: #DFE7EF;  /* 冬の窓辺 */
  --winter-blue-400: #8FA6C4;
  --paper-100:       #F4ECD8;  /* 古紙のしおり */
  --lamp-500:        #E8B45F;  /* 電灯のあかり */
}
```

## 4. アイコン体系

- **植物**: ボタニカルライン画（冬の野花・椿・南天）
- **道具**: お茶・本・電灯・しおり・絵手紙・万年筆
- **動物**: つながるマスコットを「室内ポーズ」で

## 5. トーン

- 文体: つながると同じ柔らかい敬語、ただし**少し詩的**「冬の朝ですね」「ここで一息どうぞ」
- フォント: つながると同じNoto Sans JP（明朝にはしない）

## 6. メタファー対比

| つながる | PHRシリーズ（PHR） |
|---|---|
| 庭・木漏れ日 | 縁側・冬の日溜まり |
| 外で出会う | 家で休む |
| 一緒に夜明けを見る | 一緒に温かい部屋にいる |
| 紙のランタン | 電灯・お茶の湯気 |
| 春の朝 | 冬の朝 |

## 7. リスクと対策

- **リスク**: 案Aと差が薄く見えて、結局統一でいいじゃないかとなる
- **対策**: 「冬の室内」というシーン固定で世界観の差を出す

## 8. 効果

- ファンには「同じよりそいの兄弟だ」と認識される
- 医療パスポート的な信頼感も、お茶や本のメタファーで担保
- ターゲット（重い疾患の寛解期）の「家にいる時に開く」前提とマッチ
