# よりそい てちょう — 医療パスポート

## 1. 一言で

> **診察前の「あれ何だっけ」に備える、多科併診のひとつの手帳。**

紙のお薬手帳と医師向けサマリーを置き換える。
他科・他クリニックに患者が自分で説明する場面を支える。
医療従事者に「これは便利」と言ってもらえる正統派サマリー。

## 2. 患者の課題

| 課題 | 場面 | てちょうの答え |
|---|---|---|
| **多科併診で情報が散らばる** | 「血液内科では○○、脳神経内科では××」 | クリニック軸タブ・全科横断タブで切り替え可能 |
| **薬の名前を覚えていない** | 救急・初診時に「飲んでいる薬」を聞かれる | 薬名・用量・処方医を一画面で提示 |
| **検査値の変化が分からない** | 「CRPが上がってるけど、最近どう？」 | 時系列グラフ・「炎症の数値（CRP）」とふりがな |
| **緊急時に医師に何を伝えるか** | 救急受診時の混乱 | あんしんカード（緊急情報の1枚）が一発で見せられる |

## 3. 3つのビュー

### 3-1. 全科横断ビュー（デフォルト）

すべての診療科の現在処方・最新検査値・直近イベントを1ページに圧縮。
紙のお薬手帳の発展版として、患者本人も「自分の状態を客観的に伝える」のに使える。

### 3-2. このクリニック向けビュー

「これから血液内科に行く」と選ぶと、その科に関連する過去履歴・現在処方・気になっている症状だけが表示される。
複数科併診者の「他の科の情報も持っていくと医師が困る」問題を解決。

### 3-3. あんしんカード（緊急時用）

既存 `emergency-card.html` を継承して「あんしん」にリブランド済み。
緊急連絡先・現在処方・既往歴・アレルギー・主治医情報を1画面で。
QRコード生成で家族・救急隊への共有も可能。

## 4. デザイン原則

- **略語を出さない**: 「CRP」ではなく「炎症の数値（CRP）」
- **大きい字**: 高齢の家族にも見せられる
- **印刷・スクショ前提**: 紙でも持ち運べる
- **コピーしやすい**: 主治医にLINEで送る・他科の医師にもサクッと共有

## 5. ターゲット患者像

- **コア**: 多科併診者（POEMS・SLE・パーキンソン・骨形成不全症・複数疾患持ち高齢者）
- **特徴**: 紙のお薬手帳ユーザー・救急・初診で困った経験のある人
- **拡張**: 「医療パスポート」として全患者に有用

## 6. 既存資産との対応

| 既存 | てちょうでの扱い |
|---|---|
| `public/simple/show.html`（みせる） | 全科横断ビュー・このクリニック向けビューのベース |
| `public/simple/calendar.html`（ふりかえる） | タイムライン視点として併設 |
| `public/emergency-card.html` → `anshin-card.html` | あんしんカードとしてリブランド済み |
| `server/routes/visits.js` `clinics.js` `medications.js` `lab-tracker.js` | データ層継続 |
| 用語辞書 `data/term-glossary.json` | 略語→患者向け言い換え |

## 7. 主要画面（image生成候補）

1. **ホーム** — 「今日はどの科に行きますか？」or「全科をまとめて見る」
2. **全科横断ビュー** — 美しいタイムライン・薬・検査値が見やすく整理
3. **このクリニック向けビュー** — 選択した科だけの情報・コピーボタン
4. **あんしんカード** — QRコード付き・緊急情報1画面
5. **印刷プレビュー** — 紙で持ち歩く時の見え方

## 8. 設計判断（Why）

- **「みせる」→「てちょう」にリブランドした理由**: 「みせる」は他者向けの仕事感が強い。「てちょう」は自分のために持つもの・親しみやすい
- **多科併診者を主軸にした理由**: 既存PHRの一番の差別化要素。Apple Health等の単科前提アプリでは解決できない
- **「あんしんカード」リブランドの理由**: 「緊急カード」は不安を煽る。「あんしん」は備えの心強さを示す
- **印刷・スクショ前提**: ベテラン患者は紙のお薬手帳に慣れている。デジタル一辺倒では使ってもらえない

## 9. やらないこと

- 医師の業務効率化を主目的にしない（系統A=hikariに任せる）
- 患者が見ない医療用語の生表示（必ず患者向け言い換え）
- 「全部見せる」過剰サマリー（医師が圧倒される）
