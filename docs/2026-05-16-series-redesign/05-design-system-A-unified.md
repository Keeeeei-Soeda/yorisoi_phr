# PHRシリーズ デザインシステム — 案A: 「つながる」と世界観統一

## 0. 方針

「つながる」と同じピーチ×抹茶クリーム×ラベンダーの世界観を**そのまま**採用する。
キャラ・メタファーも共有して、「よりそい」シリーズとしてのファミリー感を最大化する。

## 1. 世界観

- **メタファー**: 庭・木漏れ日・小さな森の動物たち（つながると同一）
- **時間帯**: 夜明けの少し後・朝の光
- **温度感**: あたたかい・絵本のような・絵手紙のような

## 2. キャラクター（つながると共有）

| キャラ | 役割（PHRでの） |
|---|---|
| 🌱 もか（うさぎ） | 入口AI・聞き役・状態判定の案内役（つながると同じ） |
| 🍞 ぱお（くま） | きろくの空状態・記録を見守る役 |
| 🌿 そら（猫） | そうだんの会話相手・気まぐれに気づきを返す |
| 🌸 ふう（小鳥） | てちょうのお便り運び役・通知 |

## 3. カラー

つながると同一の CSS variables を採用（`docs/design-system-tsunagaru.md` を共有）。
- `--peach-600: #E66A45`（主要ボタン）
- `--matcha-600: #6E8F4F`（成功・成長）
- `--lavender-600: #8E7AA0`（静かな夜・休息）
- 背景クリーム `--bg-cream: #FFFBF5`

## 4. トーン

- 文体: つながると同じ「○○ですよ」「○○してくれましたね」のやわらかい敬語
- アイコン: 線画+水彩・絵本タッチ
- 余白: たっぷり

## 5. PHRシリーズ独自の差別化点（最小）

つながると統一しつつ、PHRシリーズだけの軽い変化：
- **きろく**: 動物が「ノート/手帳」を抱える絵
- **そうだん**: 動物が「お茶」を持って隣に座る絵
- **てちょう**: 動物が「絵地図」を持っている絵

## 6. リスクと対策

- **リスク**: 「医療パスポート」的な信頼感が薄くなる可能性
- **対策**: てちょうのアイコンだけは線が引き締まったクラフト感を入れる

## 7. 効果

- 「よりそい」シリーズとしてのブランド感が最大
- ファンが両方使うときの世界観移行がなめらか
- 開発速度: つながるのデザイントークン流用で最速
